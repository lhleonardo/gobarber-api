# gobarber-api

API da aplicação GoBarber, desenvolvida durante o bootcamp GoStack gerenciado pela Rocketseat

# Especificações

As características técnicas são descritas abaixo:

- Node.js
- Typescript
- Express
- TypeORM
- JWT
- PostgreSQL
- RESTful
